# 논리연산 정리

## 1. NOT
- 의미: 참 ↔ 거짓을 뒤집는다.
- 예:
  - NOT True  → False
  - NOT False → True
---

## 2. AND
- 의미: 둘 다 참일 때만 True
- 조건:
  - True AND True → True
  - 그 외 → False

---

## 3. OR
- 의미: 하나라도 참이면 True
- 조건:
  - False OR False → False
  - 그 외 → True
---

## 4. XOR (배타적 논리합)
- 의미: 참/거짓이 서로 다를 때만 True
- 조건:
  - True XOR False → True
  - False XOR True → True
  - True XOR True → False
  - False XOR False → False


- 정리 공식:

  XOR → a != b


- 논리식 전개 공식:

  (a and not b) or (not a and b)
