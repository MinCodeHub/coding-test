⭐ 코테에서 진짜 많이 쓰는 문법

1. 빠른 입력

```python 
arr = list(map(int, input().split()))
```
예시 입력
```
5
1 2 3 4 5
```

결과
```
arr = [1,2,3,4,5]
```

2. 리스트 컴프리헨션

for문 한 줄로 쓰기

arr = [i for i in range(5)]

결과

[0,1,2,3,4]

조건도 가능

even = [i for i in range(10) if i % 2 == 0]


3. 빠른 탐색 set

```python 
s = set(arr)

if 3 in s:
    print("있음")
```

4. 정렬

```python 
arr.sort()
```
또는

```python 
sorted_arr = sorted(arr)
```

내림차순

```python 
arr.sort(reverse= True)
```

5. 딕셔너리 : 빈도수 계산에 많이 사용

```python 
d = {}
d["apple"] = 3
d["banna"] = 5
```

사용

```python
print(d["apple"])
```

예시

```python
for x in arr:
    d[x] = d.get(x,0)+1
```
6. 스택/큐

스택

```python
stack = []
stack.append(1)
stack.append(2)

stack.pop()
```

큐(BFS 필수)

```python
from collections import deque

q = deque()

q.append(1)
q.popleft()

```

7. 이분탐색

코테에서 수 찾기
숫자 카드
lower bound 같은 문제에 사용

```python
import bisect

arr.sort()
idx = bisect.bisect_left(arr, target)
```

8. 문자열 뒤집기

```python
s[::-1]
```

9. enumerate(인덱스 + 값 동시에)
